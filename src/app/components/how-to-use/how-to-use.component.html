How to use

  • 1 Download CSV file from Draft Kings for the contest you want to generate lineups for
  • 2 Visit the Infinity Draft permuter and fill required fields in
  • 3 Upload CSV file
  • 4 Click 'Permute' and wait as lineups are generated!

The Score metric is the sum of the Average Fantasy Points Per Game for each player on the team.

If you like a lineup, you can click Copy Ids which copies a CSV string to your clipboard. It can then be pasted into an excel file to be imported into Draft Kings.
